- Pour que le code apparaisse correctement, assignez les tabulations à une longueur de 4

- Pour lancer le programme, allez sur votre IDE préféré et lancez la fonction main depuis le fichier src/bataille/Game.java

- Les bateaux doivent être placés au format suivant:
A5 e

A5 sont les coordonnées sur la grille, et 'e' est la direction:
	n: nord
	s: sud
	e: est
	w: ouest

- Il suffit d'entrer les coordonnées sur la grille lorsqu'on vous demande "où frapper ?"
